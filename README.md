# Reindeer Exercise

## Description
Getting an array of colors and an array of reindeer to spit out on the DOM when combined. 
## How to run
In your terminal run: 
```
http-server -p 9999
```
Then navigate to [http://localhost:9999](ttp://localhost:9999) in your browser
